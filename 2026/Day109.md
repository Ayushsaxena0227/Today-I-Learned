1. What is Strapi?
Strapi is an open-source Headless CMS that allows developers to create APIs and manage content through an admin dashboard without building CRUD functionality manually.

It automatically generates REST and GraphQL APIs for content models.

2. What is a Headless CMS?
A Headless CMS manages only content and exposes it through APIs.
Unlike traditional CMS systems like WordPress, it does not control frontend rendering.
Frontend can be built using:

React
Next.js
Vue
Angular
Mobile Apps
3. Why use Strapi?
Benefits:

Faster development
Auto-generated APIs
Admin dashboard
Role-based access control

REST and GraphQL support
Easy content editing by non-technical users

5. What database does Strapi support?
Strapi supports:

PostgreSQL
MySQL
MariaDB
SQLite

SQLite is commonly used for development.

PostgreSQL is preferred in production.

Components are reusable groups of fields.

Example:

Address
 ├─ City

 └─ Country

Can be reused across multiple content types.
